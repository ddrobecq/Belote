# Belote
This is a web app to calculate scores when playing to Belote card game

# Getting Started


# Generate PWA asset
```bash
npx pwa-asset-generator ./icon.jpg ./pwa-asset -b "#00101a" -p "0%" -m ./pwa-asset/site.webmanifest -i ./pwa-asset/index.html -q 100 -f -w -d -x 
```
